# Exercice 3 : Length of Last Word

Given a string s consisting of words and spaces, return the length of the last word in the string.

# Example 1:

    Input: s = " ab bc cd de ef fg gh h"
    Output: 1

# Example 2:

    Input: s = "Donald Trump"
    Output: 5

# Example 3:

    Input: s = "Call of duty "
    Output: 4
